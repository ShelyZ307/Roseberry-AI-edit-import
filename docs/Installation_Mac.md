# Mac Installation

Install location:

```text
~/Roseberry/AI Edit Import
```

Resolve launcher location:

```text
~/Library/Application Support/Blackmagic Design/DaVinci Resolve/Fusion/Scripts/Utility/Roseberry AI Tools/Roseberry AI Edit Import.py
```

Steps:

```bash
cd roseberry-ai-edit-import-github-ready
./scripts/setup_mac.sh
./scripts/install_launcher_mac.sh
```

Then restart DaVinci Resolve and open:

```text
Workspace > Scripts > Utility > Roseberry AI Tools > Roseberry AI Edit Import
```

Manual update:

```bash
./scripts/update_mac.sh
```

Phase 1 updates require the GitHub manifest URL to be filled in first.
