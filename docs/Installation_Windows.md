# Windows Installation

Install location:

```text
%LOCALAPPDATA%\Roseberry\AI Edit Import
```

Resolve launcher location:

```text
%APPDATA%\Blackmagic Design\DaVinci Resolve\Support\Fusion\Scripts\Utility\Roseberry AI Tools\Roseberry AI Edit Import.py
```

Steps:

```bat
scripts\setup_windows.bat
scripts\install_launcher_windows.bat
```

Then restart DaVinci Resolve and open:

```text
Workspace > Scripts > Utility > Roseberry AI Tools > Roseberry AI Edit Import
```

Manual update:

```bat
scripts\update_windows.bat
```

Phase 1 updates require the GitHub manifest URL to be filled in first.
